# Synqra: Sync for Extra

AI Personal Agent App

## Getting Started

First, run the development server:

```bash
npm install
# then
npm run dev
# don't forget to add environment variables
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can check out [my GitHub repository](https://github.com/OtakuTotipotent/) - your feedback and contributions are welcome!
